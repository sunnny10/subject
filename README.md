### 관련 개념

|Repository|Git으로 관리하는 프로젝트 저장소. 소스코드 및 파일을 저장하는 일종의 폴더를 의미한다.|
|branch|독립적으로 어떤 작업을 진행하기 위한 개념. 각자 독립적인 작업 영역(저장소) 안에서 자유롭게 소스코드 변경이 가능하다.|
|commit|파일을 추가하거나 변경한 내용을 저장소에 저장하는 작업.|
